Final Project — Assignment 8

Wooden Wallet — Walnut wallet with 3D Printed PLA initials
 

Final Project

For my final project I used a combination of OnShape, Rhino, Interlocking Parts, CNC Milling, Laser Cutter, and the 3D Printer to make a wooden wallet with slightly rounded edges and initials as a birthday present for a friend. To create this I started with OnShape to design all the sides of the wooden wallet so they will press fit together. Once I was finished I exported the shapes as DXF so I could open them in Illustrator. From Illustrator I was able to send my files to the Laser Cutter. After making a couple adjustments to the file measurements each piece press fit together to make the wallet. Next I made two files for the initials RM in Illustrator. One file was essentially a thicker version of the initials and one was a thin version of the initials. The thick version would be used to cut a toolpath of the letters halfway through the walmut wood on the Bantam CNC Mill. The thin version would be printed out on a 3D Printer and later inlayed into the milled recession. Lastly I sanded the wallet and finished it off with an oil rub.

Special Thanks to Andrew Waer for CNC Milling and 3D Printing help.

 

Steps:

Build Wallet Shapes in Onshape and export shapes as DXF Files
 
 

Open DXF of Front of Wallet in Illustrator and print on Laser Cutter

Laser Cutter settings: Speed - 15,  Strength, - 100 Frequency - 50

 

Open DXF of Back of Wallet in Illustrator and print on Laser Cutter

Laser Cutter settings: Speed - 15, Strength, - 100 Frequency - 50

 

Open DXF of Bottom and Sides of Wallet in Illustrator and print on Laser Cutter

Laser Cutter settings: Speed - 15, Strength, - 100 Frequency - 50

 
Create initals with added stroke for toolpathing the recession
 

Bring the initials in to Rhino and recess them 2mm out of shape that matches the dimensions of the wallet front 2.25" x 3.625" x .127" and export STL

 
Bring in to Fusion to create toolpath
 
Set of SetUp
 
Set SetUp Stock Settings
 
Set Adaptive Settings: 1/32" Flat Mill Tool
 
Set Adaptive Settings Passes: Rought Step Down .39624mm and export GCode
 
Open GCode in Bantam Tools
 
Mill Initials. I ended up running the toolpath twice. Once with a 1/32" flat head mill and then again with a 1/64" flat head mill using the same toolpath
 
Create thin version of letters. To do this I measured the current height of the Illustrator file (14.5mm) and subtracted the height I wanted it to be (13mm) and from the height of my caliper measurements (1.5mm) Then I doubled this number (3mm) and added that much stroke to the letters. Then I selected the letters and outlined the stroke under Object/Path/Outline Stroke. This creates several paths that make up the excess letter that you can delete until you have the size letter you desire.
 
Import the letters in to Rhino. Once imported I measured the depth of my recession (2mm) created on the mill, confirmed the length letters in Rhino, and  extruded the shape to 1.875mm. I put a cap on it and exported the .stl file so I could open it in Dremel DigiLab 3D Slicer.
 
Import in to Dremel DigiLab 3D Slicer. Set Material to PLA, Profile to High Quality and Save File to USB to be read on 3D Printer
 
Load in Hatchbox PLA and set custom settings to 220° for the Nozzle Temp and 35° for the Platform Temp
 
Print Letters
 

Assemble Box

 
Glue in letters using Super Glue
 

Sand edges

 
Oil using OSMO TopOil High Solid 3056 Clear Matte
